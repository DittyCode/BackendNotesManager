### Notes Manager

# Technology

- Express
- Mongoose
- NodeJS

# Tools

- Dotenv
- Morgan
- Postman

# How to use

# ROUTE {api/v1/notes}

1. Get all notes
2. Create new note

# ROUTE {api/v1/notes/:id}

- Mongoose ID document is required

1. Get single note
2. Update note

- You can specify which data should be update {Name,Date,Complete}

3. Delete note
